
## NextJs Boilerplates

This is the nextjs boilerplate that I use for all of my NextJs projects.

```bash
yarn
yarn dev
```

Storybook

```bash
yarn storybook
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Dependencies

- storybook
- husky
- typescript
- git hook
- lint-staged
- jest
